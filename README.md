# Agentic Post Studio

An AI-powered content generation and transformation system built with LangChain and LangGraph.

## Features

- Multi-step content generation pipeline
- Customizable transformation workflows
- Graph-based processing architecture

## Quick Start

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python main.py
```

## Project Structure

- `backend/` - Core application logic
  - `src/chains/` - LangChain generation and transformation chains
  - `src/workflows/` - LangGraph workflow orchestration
  - `src/config/` - Configuration and prompts
